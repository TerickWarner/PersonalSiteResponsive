Big Beef Tracker

Python

A Big Beef, a phrase popularized by the No Dunks podcast, is when an NBA player grabs 20 or more rebounds in a single game. This program uses the NBA_API to fetch and use player statistics found on the NBA’s website to track individual rebound statistics. The program also has the ability to tweet on the account @bigbeefbot on Twitter, but is not currently active.

The Spanish Flashcard program was created to help someone study the most popular spanish words and phrases. By using python’s built in module, tkinter, I created a graphical user interface that allows you to keep track of words you still need to study.
